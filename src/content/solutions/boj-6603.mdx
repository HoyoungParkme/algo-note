## 문제 요약

K개의 수에서 **6개를 골라** 만들 수 있는 모든 조합을 사전순으로 출력한다.
입력은 여러 줄, K=0이면 종료.

## 접근

`itertools.combinations`로 6개 조합을 모두 생성. 입력이 이미 정렬되어 들어오므로 출력도 자동으로 사전순이다.

- 조합의 개수: C(K, 6). K ≤ 13이므로 최대 C(13, 6) = 1716개

## 풀이 코드

```python
import sys
from itertools import combinations
input = sys.stdin.readline

while True:
    line = list(map(int, input().split()))
    K = line[0]
    if K == 0:
        break
    nums = line[1:]
    for combo in combinations(nums, 6):
        print(*combo)
    print()                    # 각 테스트 케이스 사이 빈 줄
```

## 핵심 포인트

- **`combinations(iterable, r)`**: 순서 무관, 사전순으로 자동 생성 (입력이 정렬돼 있으면)
- **`print(*combo)`**: 튜플 언패킹으로 공백 구분 출력
- 입력 형식이 `K n1 n2 ... nK`로 한 줄 → 첫 원소가 K, 나머지가 수
- K=0이 종료 신호 → `while True + break` 패턴
