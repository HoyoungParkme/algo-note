## 문제 요약

N×N 체스판에 N개의 퀸을 서로 공격할 수 없도록 배치하는 경우의 수를 구한다. 퀸은 같은 행/열/대각선의 어떤 칸도 공격할 수 있다. (N ≤ 14)

## 접근

백트래킹의 정형. **한 행에 퀸 하나씩** 두는 결정을 내리고, 조건 위반 시 즉시 되돌아간다.

- 같은 행에 두 개를 둘 일은 없으므로 행 기준 진행
- 같은 열, 같은 우대각선(`row + col`), 같은 좌대각선(`row - col + N`)에 퀸이 있으면 가지치기
- 모든 행을 다 채우면 카운트 +1

## 풀이 코드

```python
import sys
input = sys.stdin.readline

N = int(input())

col_used = [False] * N
diag1 = [False] * (2 * N)       # row + col
diag2 = [False] * (2 * N)       # row - col + N (음수 방지)

def solve(row: int) -> int:
    if row == N:
        return 1
    count = 0
    for col in range(N):
        if col_used[col] or diag1[row + col] or diag2[row - col + N]:
            continue
        col_used[col] = diag1[row + col] = diag2[row - col + N] = True
        count += solve(row + 1)
        col_used[col] = diag1[row + col] = diag2[row - col + N] = False
    return count

print(solve(0))
```

## 핵심 포인트

- **가지치기 3종**: 열 / 우대각선(`row+col`) / 좌대각선(`row-col`)
- **대각선 인덱스**: `row+col`은 우↘ 대각선 (값 같음), `row-col`은 좌↙ 대각선 (값 같음). 음수 방지로 N을 더함
- **백트래킹 패턴**: 선택 → 재귀 → 되돌리기 (3단계 고정)
- N=14에서도 1초 내 (가지치기 효과). 가지치기 없으면 N!=87 billion으로 폭발
