## 문제 요약

1번부터 N번까지 N명의 사람이 원형으로 앉아 있다. K번째 사람을 차례대로 제거할 때, 제거된 사람의 순서를 출력한다.

## 접근

`deque`의 `rotate`와 `popleft`로 원형 큐를 시뮬레이션한다.

- 1~N을 deque에 넣고
- `rotate(-(K-1))`로 K번째 사람을 가장 앞으로 회전
- `popleft()`로 제거 + 결과 저장
- 큐가 빌 때까지 반복

`rotate(n)`은 양수면 오른쪽으로(앞 → 뒤), 음수면 왼쪽으로(뒤 → 앞) 회전한다.

## 풀이 코드

```python
import sys
from collections import deque
input = sys.stdin.readline

N, K = map(int, input().split())

q = deque(range(1, N + 1))
result = []

while q:
    q.rotate(-(K - 1))   # K번째 사람을 맨 앞으로
    result.append(q.popleft())

print('<' + ', '.join(map(str, result)) + '>')
```

## 핵심 포인트

- **rotate 부호**: 음수 = 왼쪽 회전 = 앞에서 뒤로 이동
- `K-1`을 회전하는 이유: 처음 사람이 0번째 위치에 있으므로 K번째는 K-1만큼 이동해야 맨 앞에 옴
- 출력 형식 `<a, b, c>` 주의 — `, `(콤마+공백)으로 join
- N, K가 작으므로 O(N·K)로 충분
