## 문제 요약

수열 A가 주어질 때, 가장 긴 **증가하는 부분 수열**(LIS)의 길이를 구한다. 부분 수열은 연속하지 않아도 됨.

## 접근

`dp[i]` = `arr[i]`로 끝나는 LIS의 길이.

- 모든 i에 대해 dp[i] = 1로 초기화 (자기 자신만 포함)
- i 이전의 모든 j를 보면서, `arr[j] < arr[i]`이면 dp[i]를 `dp[j] + 1`로 갱신 가능
- 답은 `max(dp)`

O(N log N) 해법(이분탐색 + tails 배열)도 있지만 N=1000이면 O(N²)로 충분.

## 풀이 코드

```python
import sys
input = sys.stdin.readline

N = int(input())
arr = list(map(int, input().split()))

dp = [1] * N
for i in range(1, N):
    for j in range(i):
        if arr[j] < arr[i]:
            dp[i] = max(dp[i], dp[j] + 1)

print(max(dp))
```

## 핵심 포인트

- **LIS의 정의 차이**: "증가" vs "비감소". `<` vs `<=`로 구분
- **dp[i] 정의를 정확히**: "i로 끝나는 LIS"이지 "0~i 중 LIS"가 아님 → max(dp) 필요
- N ≤ 1000 → O(N²)=10⁶ 안전. N ≤ 1,000,000이면 이분탐색 LIS 필요 (BOJ 12015)
- 같은 패턴: 가장 긴 감소 부분 수열은 부등호만 뒤집으면 됨
