<Why>
- 문자열/데이터를 고정 크기 숫자로 → **O(1) 조회**
- Python dict·set이 내부적으로 해시 사용
- "중복 확인", "빈도 세기", "패턴 매칭"에 활용
</Why>

<How>
- 같은 입력 → 항상 같은 해시값
- 다른 입력 → 같을 수 있음 = **해시 충돌**
</How>

```python
from collections import Counter, defaultdict

# ─── 등장 횟수 세기 (BOJ 10816 숫자 카드 2) ───
cnt = Counter(arr)                # {값: 횟수} 딕셔너리
print(cnt[x])                     # x의 등장 횟수 O(1)

# ─── 중복 확인 ───
seen = set()
for x in arr:
    if x in seen:                 # O(1) 조회 (리스트 in은 O(N)!)
        print("중복!")
    seen.add(x)

# ─── 두 배열의 공통 원소 ───
set_a = set(arr_a)                # set으로 변환
common = [x for x in arr_b if x in set_a]  # O(N)

# ─── defaultdict (키 없어도 에러 안 남) ───
graph = defaultdict(list)         # 없는 키 접근 시 빈 리스트
count = defaultdict(int)          # 없는 키 접근 시 0

# ─── 문자열 해시 (롤링 해시) ───
BASE = 31; MOD = 1_000_000_007
def string_hash(s):
    h = 0
    for c in s:
        h = (h * BASE + ord(c) - ord('a') + 1) % MOD
    return h
```
