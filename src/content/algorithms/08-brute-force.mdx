<Why>
- N이 작으면(≤20) **전부 보는 게 가장 확실**
- 최적해를 놓칠 걱정 없음
- DP/그리디가 안 떠오를 때 일단 완전탐색으로 정답 확인 후 최적화
</Why>

<How>

|:--|:--|:--|
| 조합 | 순서 무관 | C(N,R) |
| 순열 | 순서 중요 | P(N,R) |
| 부분집합 | 뽑거나 안 뽑거나 | 2^N |
</How>

from itertools import combinations, permutations

# ─── 조합: N개에서 R개 뽑기 (순서 무관) ───
# BOJ 6603 로또, 15686 치킨 배달
for combo in combinations(arr, r):
    if check(combo):                  # 각 조합마다 조건 체크
        answer = max(answer, calc(combo))

# ─── 순열: N개에서 R개 뽑기 (순서 중요) ───
# BOJ 10819 차이를 최대로
for perm in permutations(arr):
    score = sum(abs(perm[i]-perm[i+1]) for i in range(len(perm)-1))
    answer = max(answer, score)

# ─── 부분집합: 비트마스크 ───
