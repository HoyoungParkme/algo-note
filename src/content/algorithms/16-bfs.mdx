<Why>
- 가중치 없는 그래프에서 **최단 거리 보장**
- "최소 이동 횟수", "최단 경로" → 가장 먼저 떠올려야
- 2D 격자(미로) BFS → 코테 단골
</Why>

<How>
큐(deque) 사용. 이웃을 큐에 넣고 하나씩 꺼내며 탐색. 방문 체크 필수.
</How>

### 그래프 BFS — 인접 리스트 패턴

```python
from collections import deque
def bfs(start):
    visit = [False]*(N+1)
    dist = [-1]*(N+1)               # -1 = 미방문
    q = deque([start])
    visit[start] = True; dist[start] = 0

    while q:
        u = q.popleft()              # FIFO: 가까운 노드부터
        for v in adj[u]:
            if not visit[v]:
                visit[v] = True
                dist[v] = dist[u]+1  # 이전 거리 + 1
                q.append(v)
    return dist
```

dx = [-1,1,0,0]  # 상하좌우 이동
dy = [0,0,-1,1]

def bfs_grid(sx, sy):
    dist = [[-1]*M for _ in range(N)]  # -1: 미방문
    dist[sx][sy] = 1                    # 시작점 거리 1 (문제 조건)
    q = deque([(sx, sy)])

    while q:
        x, y = q.popleft()
        for d in range(4):                # 4방향 탐색
            nx, ny = x+dx[d], y+dy[d]
            # 범위 체크 + 벽 아닌지 + 미방문인지

### 토마토 (BOJ 7576) — 다중 시작점 BFS

# 익은 토마토 위치를 전부 큐에 넣고 시작
q = deque()
for i in range(N):
    for j in range(M):

### 좌표 집합(set) 기반 BFS — 격자 대신 좌표만 주어질 때

격자가 sparse하거나 좌표만 K개 입력으로 주어질 때, 2D 배열을 만들지 않고 `set`에 좌표를 저장한 뒤 멤버십 체크로 BFS를 돌린다. 가장 큰 연결 영역 크기를 구하는 패턴.

```python
from collections import deque

N, M, K = map(int, input().split())
points = set()
for _ in range(K):
    x, y = map(int, input().split())
    points.add((x, y))

visited = set()
max_size = 0

def bfs(start):
    q = deque([start])
    visited.add(start)
    size = 1
    while q:
        x, y = q.popleft()
        for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
            nx, ny = x + dx, y + dy
            if (nx, ny) in points and (nx, ny) not in visited:
                visited.add((nx, ny))
                q.append((nx, ny))
                size += 1
    return size

for p in points:
    if p not in visited:
        max_size = max(max_size, bfs(p))

print(max_size)
```

**격자 BFS와의 차이**:

| | 격자 BFS | 좌표 set BFS |
|:--|:--|:--|
| 길 체크 | `grid[nx][ny] == 1` | `(nx, ny) in points` |
| 범위 체크 | `0 <= nx < N and 0 <= ny < M` | 불필요 (set에 없으면 자동 제외) |
| 방문 체크 | `dist[nx][ny] == -1` 또는 `visit` 배열 | `(nx, ny) not in visited` |
| 적합한 상황 | 격자 크기 ≤ 1000, dense | 좌표 범위 매우 큼(10⁹+), 점이 sparse(K개) |
| 메모리 | O(N · M) | O(K) |

<Tip title="좌표를 hash 키로">tuple `(x, y)`는 hashable이라 set/dict의 키로 바로 쓸 수 있음. 리스트 `[x, y]`는 hashable 아니므로 절대 사용 X.</Tip>
