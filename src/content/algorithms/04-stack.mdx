<Why>
- **LIFO**(Last In, First Out) — 가장 최근 것을 먼저 처리
- "괄호 검사", "뒤로 가기", "DFS", "후위 표기식", "단조 스택" 등에 사용
- Python 리스트로 바로 구현: `append()` + `pop()`
</Why>

<How>
- `stack.append(x)` — push O(1)
- `stack.pop()` — pop O(1)
- `stack[-1]` — peek O(1)
- `len(stack)==0` 또는 `not stack` — 비어있는지
</How>

### 괄호 검사

<SolutionBlock problemId="boj-9012">

```python
def is_valid(s):
    stack = []
    pair = {')':'(', ']':'[', '}':'{'}  # 닫는→여는 매핑
    for c in s:
        if c in '([{':              # 여는 괄호 → push
            stack.append(c)
        elif c in pair:             # 닫는 괄호
            if not stack or stack[-1] != pair[c]:
                return False        # 짝이 안 맞음
            stack.pop()             # 짝 맞으면 pop
    return len(stack) == 0          # 남은 괄호 없어야 성공
```

### 단조 스택 — 오른쪽에서 처음으로 더 큰 값 (NGE)

```python
# BOJ 17298 오큰수, 탑 등에서 활용
answer = [-1] * N           # 기본값: 없으면 -1
stack = []                  # 아직 오큰수를 못 찾은 인덱스
for i in range(N):
    # 현재 값이 스택 top보다 크면 → 스택 top의 오큰수가 현재 값
    while stack and arr[stack[-1]] < arr[i]:
        answer[stack.pop()] = arr[i]
    stack.append(i)         # 현재 인덱스를 스택에 push
```
