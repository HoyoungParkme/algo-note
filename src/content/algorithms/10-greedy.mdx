### 회의실 배정 (BOJ 1931) — 끝나는 시간 기준

<SolutionBlock problemId="boj-1931">

```python
# 핵심: 일찍 끝나는 회의부터 선택 → 가장 많이 배정
meetings.sort(key=lambda x: (x[1], x[0]))  # 끝나는 시간 오름차순
end = count = 0
for s, e in meetings:
    if s >= end:              # 이전 회의 끝난 후 시작 가능하면
        count += 1            # 이 회의 선택
        end = e               # 끝나는 시간 갱신
```

</SolutionBlock>

### ATM 대기시간 최소 (BOJ 11399) — 짧은 순서

```python
# 핵심: 시간 짧은 사람부터 처리 → 전체 대기시간 최소
times.sort()                  # 오름차순 정렬
total = wait = 0
for t in times:
    wait += t                 # 누적 대기시간
    total += wait             # 전체에 누적
```

<Warn>반례: 동전 [1,6,10], 12원 → 그리디(10+1+1)=3개, 최적(6+6)=2개</Warn>
