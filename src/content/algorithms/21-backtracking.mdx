<Why>
- 완전탐색인데 **가지치기(pruning)** 로 불필요한 경우를 미리 잘라냄 → 시간 대폭 절약
- "조건을 만족하는 모든 조합을 구하라" 유형에서 핵심
- DFS + 조건 체크 + 되돌리기가 기본 구조
</Why>

<How>
- DFS로 한 단계씩 선택을 진행
- 현재 선택이 **조건에 안 맞으면 즉시 돌아감** (가지치기)
- 조건에 맞으면 다음 단계로 재귀
- 재귀가 끝나면 선택을 **취소(되돌리기)** 하고 다른 선택 시도

### 기본 뼈대

```python
def backtrack(depth, state):
    # 종료 조건: 원하는 깊이에 도달
    if depth == target:
        result.append(state[:])     # 현재 상태 저장 (복사!)
        return

    for choice in candidates:
        # ─── 가지치기: 조건 안 맞으면 스킵 ───
        if not is_valid(choice):
            continue                # 이 가지는 포기 (pruning)

        # ─── 선택 ───
        state.append(choice)
        used[choice] = True         # 사용 체크

        # ─── 다음 단계로 재귀 ───
        backtrack(depth + 1, state)

        # ─── 되돌리기 (선택 취소) ───
        state.pop()
        used[choice] = False        # 사용 해제
```

### N-Queen (BOJ 9663)

```python
# N×N 체스판에 N개의 퀸을 서로 공격 못 하게 배치
def solve(row):
    if row == N:                    # 모든 행에 퀸 배치 완료
        return 1

    count = 0
    for col in range(N):
        # 가지치기: 같은 열, 대각선에 퀸 있으면 스킵
        if col_used[col] or diag1[row+col] or diag2[row-col]:
            continue

        # 선택
        col_used[col] = diag1[row+col] = diag2[row-col] = True
        count += solve(row + 1)     # 다음 행으로

        # 되돌리기
        col_used[col] = diag1[row+col] = diag2[row-col] = False

    return count
```
