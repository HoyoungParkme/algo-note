<Why>
- 같은 결과의 프로그램 중 **어떤 게 더 빠른지 비교**하는 기준
- 코테 시간 제한(1~2초) 안에 통과하려면, N에 따라 알고리즘을 골라야 함
- N ≤ 100,000 → O(N log N) OK, N ≤ 20 → O(2^N)도 가능
</Why>

<How>
연산 횟수를 세고, **최고차항만 남기고 계수를 버린다**.

| 실제 연산 | 복잡도 | 이유 |
|:--|:--|:--|
| 2N+1 | O(N) | 상수·계수 제거 |
| 3N²+9N+3 | O(N²) | N² 압도적 |

```text
O(1) < O(log N) < O(N) < O(N log N) < O(N²) < O(2^N) < O(N!)
```
</How>
