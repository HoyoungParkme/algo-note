### 재귀 DFS

<SolutionBlock problemId="boj-2667">

```python
import sys; sys.setrecursionlimit(10**6)  # 재귀 제한 해제!

def dfs(u):
    visit[u] = True             # 방문 체크
    for v in adj[u]:            # 인접 노드 순회
        if not visit[v]:
            dfs(v)              # 재귀 호출 (깊이 우선)
```

### 반복문 DFS (재귀 깊이 걱정 없음)

```python
def dfs_iter(start):
    stack = [start]            # 스택에 시작점
    while stack:
        u = stack.pop()        # LIFO: 가장 최근 것부터
        if visit[u]: continue  # 이미 방문했으면 스킵
        visit[u] = True
        for v in adj[u]:
            if not visit[v]:
                stack.append(v)
```

### 연결 요소 개수 (BOJ 11724)

```python
count = 0
visit = [False]*(N+1)
for i in range(1, N+1):
    if not visit[i]:         # 아직 방문 안 한 노드 발견
        dfs(i)               # 거기서 DFS → 연결된 것 전부 방문
        count += 1           # DFS 1번 = 연결 요소 1개
```
