<Why>
- **같은 하위 문제 반복** → 결과 저장으로 중복 제거
- 완전탐색 O(2^N) → DP로 O(N), O(N²)으로 축소
- "최소/최대/경우의 수" + "순서 있는 선택" → DP 의심
</Why>

<How>
- dp[i] 정의 → 점화식 → 초기값 → 채우기 → 답 추출
</How>

### 1D DP — 계단 오르기 (BOJ 2579)

```python
# dp[i] = i번째 계단까지 최대 점수
dp = [0]*(N+1)
dp[1] = stairs[1]
dp[2] = stairs[1] + stairs[2]

for i in range(3, N+1):
    dp[i] = max(
        dp[i-2] + stairs[i],               # 2칸 점프
        dp[i-3] + stairs[i-1] + stairs[i]  # 1+1 (연속 3칸 방지)
    )
```

### 2D DP — 배낭 (BOJ 12865)

```python
# dp[i][w] = i번째 물건까지, 무게 w일 때 최대 가치
dp = [[0]*(W+1) for _ in range(N+1)]
for i in range(1, N+1):
    wi, vi = items[i]                      # 무게, 가치
    for w in range(W+1):
        if w >= wi:
            dp[i][w] = max(dp[i-1][w],     # 안 넣기
                           dp[i-1][w-wi]+vi) # 넣기
        else:
            dp[i][w] = dp[i-1][w]          # 못 넣음
```

### LIS — 가장 긴 증가 부분수열 (BOJ 11053)

dp = [1] * N                              # 자기 자신만 포함 = 1
for i in range(1, N):

        if arr[j] < arr[i]:               # 증가하는 관계면
