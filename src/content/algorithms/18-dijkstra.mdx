<Why>
- **가중치 있는** 그래프에서 최단 거리
- BFS는 가중치 동일할 때만 → 다르면 다익스트라
- "A→B 최소 비용" 유형의 표준 해법
</Why>

<How>
- dist 무한대 초기화, 시작점 0
- 우선순위 큐에서 가장 가까운 노드 꺼내 확정
- 이웃 dist 갱신 → 큐 추가
- **음수 가중치 불가**
</How>

```python
import heapq

def dijkstra(start):
    dist = [1e9]*(N+1)             # 무한대로 초기화
    dist[start] = 0                # 시작점 거리 0
    pq = [(0, start)]              # (거리, 노드) ← 거리가 앞!

    while pq:
        d, u = heapq.heappop(pq)   # 가장 가까운 노드 꺼냄
        if d != dist[u]:           # 이미 더 짧은 경로로 확정 → 스킵
            continue               # (이게 없으면 시간초과 가능!)
        for v, w in adj[u]:        # adj: (노드, 가중치)
            if dist[v] > dist[u]+w: # 더 짧은 경로 발견!
                dist[v] = dist[u]+w
                heapq.heappush(pq, (dist[v], v))  # pq: (거리, 노드)
    return dist
```
