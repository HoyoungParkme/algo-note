<Why>
- 계층 관계(조직도, 폴더, 카테고리) 표현
- 이진 탐색 트리, 세그먼트 트리의 기반
- "부모-자식", "루트 거리", "LCA", "서브트리" 문제
</Why>

<How>
저장: **부모 배열** 또는 **자식 리스트**. 보통 adj로 입력 받고 BFS/DFS로 부모 확정.
</How>

```python
# ─── 입력 → 트리 구성 (BFS로 부모 배정) ───
# BOJ 11725 트리의 부모 찾기
from collections import deque
N = int(input())
adj = [[] for _ in range(N+1)]    # 인접 리스트
parent = [0] * (N+1)              # parent[i] = i의 부모

for _ in range(N-1):              # 간선 N-1개 입력
    a, b = map(int, input().split())
    adj[a].append(b)              # 양방향 연결
    adj[b].append(a)

# BFS로 루트(1)부터 탐색하며 부모 확정
visit = [False]*(N+1)
q = deque([1]); visit[1] = True
while q:
    u = q.popleft()
    for v in adj[u]:
        if not visit[v]:
            visit[v] = True
            parent[v] = u         # v의 부모는 u
            q.append(v)
```
